# Gestor Financiero Python

Proyecto educativo para manejar finanzas personales. Permite:

- Crear cuentas de usuario.
- Registrar ingresos y gastos.
- Mostrar historial de movimientos.
- Calcular saldo actual de la cuenta.

## Estructura
- `models.py` → Clases principales (Cuenta, Movimiento, TipoMovimiento)
- `datos.py` → Funciones de carga/guardado en JSON
- `utils.py` → Funciones de UI y manejo de datos
- `main.py` → Interfaz de usuario y flujo principal

## Cómo usar
1. Instalar Python 3.10+
2. Ejecutar `main.py`
```bash
python main.py
